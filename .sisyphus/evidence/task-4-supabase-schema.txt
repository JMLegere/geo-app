SCHEMA DETAILS:
===============

profiles table:
- id (UUID, PK, FK to auth.users)
- display_name (TEXT)
- current_streak (INT, default 0)
- longest_streak (INT, default 0)
- total_distance_km (DOUBLE PRECISION, default 0)
- current_season (TEXT, default 'summer')
- created_at, updated_at (TIMESTAMPTZ)

cell_progress table:
- id (UUID, PK)
- user_id (UUID, FK to auth.users, NOT NULL)
- cell_id (TEXT, NOT NULL)
- fog_state (TEXT, default 'undetected')
- distance_walked (DOUBLE PRECISION, default 0)
- visit_count (INT, default 0)
- restoration_level (DOUBLE PRECISION, default 0)
- last_visited (TIMESTAMPTZ)
- created_at, updated_at (TIMESTAMPTZ)
- UNIQUE(user_id, cell_id)

species table:
- id (TEXT, PK)
- name (TEXT, NOT NULL)
- biome (TEXT, NOT NULL)
- rarity (TEXT, NOT NULL)
- description (TEXT)
- season_availability (TEXT[], default '{summer,winter}')
- created_at (TIMESTAMPTZ)

collected_species table:
- id (UUID, PK)
- user_id (UUID, FK to auth.users, NOT NULL)
- species_id (TEXT, FK to species, NOT NULL)
- cell_id (TEXT, NOT NULL)
- collected_at (TIMESTAMPTZ)
- UNIQUE(user_id, species_id)

RLS POLICIES:
=============

profiles:
- SELECT: auth.uid() = id
- UPDATE: auth.uid() = id
- INSERT: auth.uid() = id

cell_progress:
- SELECT: auth.uid() = user_id
- INSERT: auth.uid() = user_id
- UPDATE: auth.uid() = user_id

species:
- SELECT: true (public read)

collected_species:
- SELECT: auth.uid() = user_id
- INSERT: auth.uid() = user_id
- DELETE: auth.uid() = user_id

EDGE FUNCTION DETAILS:
======================

Function: generate-species-for-cell
Input: { cell_id: string, biome: string }
Output: { cell_id, biome, species: Species[], count: number }

Algorithm:
1. Hash cell_id to get deterministic seed
2. Use seed to shuffle species list (Fisher-Yates)
3. Return 3-5 species from the biome
4. Same input always produces same output (deterministic)

FLUTTER INTEGRATION:
====================

Configuration file: lib/core/config/supabase_config.dart
- Loads SUPABASE_URL and SUPABASE_ANON_KEY from .env
- Provides static accessors: projectUrl, anonKey, restUrl, functionsUrl
- Throws exception if credentials missing

Environment files:
- .env.example: Template with placeholder values
- .env: Actual credentials (add to .gitignore)

Usage in Flutter:
```dart

import 'lib/core/config/supabase_config.dart';

void main() async {
  await dotenv.load(fileName: '.env');
  SupabaseConfig.initialize();
  runApp(const FogOfWorldApp());
}
```
