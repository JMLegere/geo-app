## Gameplay

- [ ] Map screen — MapLibre, fog-of-war overlay, player position dot
- [ ] GPS location service — real position, accuracy threshold, simulation fallback
- [ ] Voronoi cell system — `LazyVoronoiCellService`, cell detection, visit recording

- [ ] Detection radius — cells within 1000m at least "Detected"

### Discovery & Encounters
- [ ] Daily seed system — `DailySeedService`, midnight GMT rotation, server RPC `ensure_daily_seed()`
- [ ] Deterministic encounters — `SHA-256(dailySeed + "_" + cellId)` → species list
- [ ] Stale seed guard — discoveries pause after 24h without refresh
- [ ] Offline discovery — roll encounters from cached seed, queue for server validation
- [ ] Server validation — `validate-encounter` Edge Function, rollback on mismatch
- [ ] 3 encounter slots per cell visit
- [ ] Discovery toast — species revealed on cell visit
- [ ] Rarity-scaled reveals — LC = toast, EX = full-screen ceremony
- [ ] Season-gated species — summer-only (May–Oct), winter-only (Nov–Apr), year-round

- [ ] Sanctuary screen — zoo building, placed animals
- [ ] Sanctuary placement — Ark Nova-style appeal puzzle
- [ ] Animal caretaking — feed animals, track hunger, produce orbs
- [ ] Food system — 7 food types (critter, fish, fruit, grub, nectar, seed, veg)
- [ ] Food preference per species — AI-enriched, one food type per species
- [ ] Orb production — feed animal → 1 habitat orb + 1 class orb + 1 climate orb
- [ ] Tamagotchi mechanics — animals age, express needs, interact
